# 💰 SmartCash - Dashboard Financeiro Inteligente

Uma aplicação interativa de gestão financeira pessoal desenvolvida para simplificar o controle de receitas e despesas. O projeto explora conceitos fundamentais de desenvolvimento web frontend, como lógica de programação, manipulação dinâmica do DOM e persistência de dados local.

## 🌐 Demonstração
O projeto está publicado e pode ser acessado em tempo real através do link:
👉 [Acesse o SmartCash aqui](https://li-rodrigues.github.io/SmartCash/)

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação semântica da interface.
- **CSS3**: Estilização moderna utilizando layouts responsivos com Flexbox e Grid.
- **JavaScript (ES6+)**: Lógica de negócios, manipulação do DOM e eventos.
- **LocalStorage**: Persistência de dados diretamente no navegador do usuário, garantindo que as informações não se percam ao atualizar a página.

## 🛠️ Funcionalidades Concluídas
- [x] **Interface Responsiva**: Estrutura base visualmente limpa, moderna e adaptável.
- [x] **Gestão de Movimentações**: Registro simplificado de transações financeiras (Entradas e Saídas).
- [x] **Cálculos em Tempo Real**: Atualização automática do saldo total, total de entradas e total de saídas à medida que os dados são inseridos.
- [x] **Dashboard Dinâmico**: Visualização clara dos indicadores financeiros para tomada de decisão.
- [x] **Validação de Dados**: Filtros e travas de segurança para garantir a inserção correta dos valores.
