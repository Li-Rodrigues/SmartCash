# 💰 SmartCash - Intelligent Financial Dashboard

An interactive personal finance management application developed to simplify income and expense tracking. This project explores fundamental frontend web development concepts, such as programming logic, dynamic DOM manipulation, and local data persistence.

## 🌐 Demo
The project is live and can be accessed in real-time at:
👉 [Access SmartCash here](https://li-rodrigues.github.io/SmartCash/)

## 🚀 Technologies Used
- **Semantic HTML5**: Clean and structured interface markup.
- **CSS3**: Modern styling using responsive layouts with Flexbox and Grid.
- **JavaScript (ES6+)**: Core business logic, DOM manipulation, and event handling.
- **LocalStorage**: Data persistence directly in the user's browser, ensuring data is retained even after reloading the page.

## 🛠️ Completed Features
- [x] **Responsive Interface**: Clean, modern, and adaptable visual structure.
- [x] **Transaction Management**: Simplified logging for financial transactions (Incomes and Expenses).
- [x] **Real-Time Calculations**: Automatic updates for total balance, total incomes, and total expenses as data is entered.
- [x] **Dynamic Dashboard**: Clear visualization of financial indicators for better decision-making.
- [x] **Data Validation**: Input constraints and error prevention to ensure correct values.
