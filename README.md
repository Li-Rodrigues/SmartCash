# 💰 SmartCash - Dashboard Financeiro Inteligente

Aplicação interativa de gestão financeira desenvolvida com **JavaScript Vanilla**, explorando lógica de programação, manipulação de DOM e persistência de dados local.

## 🚀 Tecnologias
- HTML5 Semântico
- CSS3 (Flexbox/Grid)
- JavaScript Nativo (ES6+)
- LocalStorage para persistência
